# Checking tools

## Local
### python with brew
```bash
# check exist command
$ brew
# list package on brew
$ brew list
# install python with brew
$ brew install python
```

### PIP
```bash
# check exist pip3 
$ pip3 list
# install virtualenv
$ pip3 install virtualenv
```
